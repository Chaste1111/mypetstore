## 🐛 常见问题排查

### 问题 1: 图片无法显示（404 错误）
**可能原因**:
- 后端服务未启动
- Vite 代理配置错误
- 图片文件不存在

**解决方法**:
1. 检查后端是否在 8080 端口运行
2. 检查 `vite.config.js` 中的代理配置
3. 确认 `src/main/webapp/images/` 目录下有对应的图片文件

### 问题 2: 图片显示但很模糊
**可能原因**:
- GIF 图片本身分辨率低
- CSS 拉伸导致失真

**解决方法**:
- 考虑使用更高分辨率的图片
- 调整 `object-fit` 属性

### 问题 3: 图片加载很慢
**可能原因**:
- GIF 文件过大
- 网络速度慢

**解决方法**:
- 压缩 GIF 文件或转换为 WebP
- 实现图片懒加载
- 使用 CDN
