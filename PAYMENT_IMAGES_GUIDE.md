# 支付功能图片资源说明

## 需要准备的图片资源

为了完整实现支付功能，需要准备以下图片文件并放置到 `frontend/public/images/` 目录下：

### 1. 支付方式图标（用于选择界面）

- **unionpay.png** - 云闪付/银行卡支付图标
  - 建议尺寸：60x60px
  - 格式：PNG（透明背景）
  - 说明：可以使用云闪付官方Logo或银行卡图标

- **wechat_pay.png** - 微信支付图标
  - 建议尺寸：60x60px
  - 格式：PNG（透明背景）
  - 说明：使用微信支付的绿色Logo

- **alipay.png** - 支付宝图标
  - 建议尺寸：60x60px
  - 格式：PNG（透明背景）
  - 说明：使用支付宝的蓝色Logo

### 2. 支付二维码图片（用于扫码支付）

- **wechat_qr.png** - 微信支付二维码
  - 建议尺寸：280x280px
  - 格式：PNG
  - 说明：可以生成一个示例二维码图片，或者使用固定的测试二维码
  
- **alipay_qr.png** - 支付宝支付二维码
  - 建议尺寸：280x280px
  - 格式：PNG
  - 说明：可以生成一个示例二维码图片，或者使用固定的测试二维码

## 临时解决方案

如果暂时没有这些图片，系统会使用默认的占位图 `/images/splash.gif`，但这会影响用户体验。

## 图片来源建议

1. **官方Logo下载**：
   - 云闪付、微信、支付宝官方网站都有品牌资源下载
   - 注意遵守品牌使用规范

2. **图标库**：
   - 可以使用 IconFont、FontAwesome 等图标库
   - 导出为PNG格式

3. **二维码生成**：
   - 使用在线二维码生成器生成测试二维码
   - 例如：草料二维码、联图网等

4. **设计工具**：
   - 使用 Photoshop、Figma 等工具设计
   - 保持风格统一

## 图片放置位置

所有图片应该放置在：
```
E:\AAA软件平台架构\实验\mypetstore\frontend\public\images\
```

## 注意事项

- 确保图片文件名与代码中引用的完全一致（区分大小写）
- 建议使用PNG格式以保持透明度
- 控制图片大小，避免过大影响加载速度
- 保持视觉风格的一致性
