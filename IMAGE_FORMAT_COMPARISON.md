## 🎨 视觉效果对比

### GIF 格式特点
✅ **优点**:
- 文件小（~12 KB）
- 加载速度快
- 支持简单动画

❌ **缺点**:
- 最多 256 色
- 画质较差
- 不支持半透明
- 细节模糊

### PNG 格式特点
✅ **优点**:
- 无损压缩
- 支持真彩色（1600万色）
- 支持 Alpha 通道（半透明）
- 画质清晰，细节丰富

❌ **缺点**:
- 文件较大（200KB - 2.4MB）
- 加载速度较慢
- 不支持动画

### 方案 3：懒加载（强烈推荐）⭐⭐⭐⭐⭐

**原理**:
- 只加载可视区域内的图片
- 滚动时才加载其他图片

**实施方法**:
```javascript
// 使用 Intersection Observer API
const observer = new IntersectionObserver((entries) => {
  entries.forEach(entry => {
    if (entry.isIntersecting) {
      const img = entry.target
      img.src = img.dataset.src
      observer.unobserve(img)
    }
  })
})

document.querySelectorAll('img[data-src]').forEach(img => {
  observer.observe(img)
})
```

**预期效果**:
- 初始加载时间减少 60-80%
- 带宽消耗大幅降低
- 用户体验显著提升

### 方案 4：响应式图片

**原理**:
- 根据屏幕尺寸提供不同分辨率的图片

**实施方法**:
```html
<img 
  srcset="
    labuladuo-small.png 480w,
    labuladuo-medium.png 768w,
    labuladuo-large.png 1200w
  "
  sizes="(max-width: 480px) 480px,
         (max-width: 768px) 768px,
         1200px"
  src="labuladuo-medium.png"
  alt="拉布拉多"
/>
```

**预期效果**:
- 移动端流量节省 50-70%
- 桌面端保持高质量
